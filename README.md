# ChronoX - Social Gaming Platform

A next-generation social platform combining social networking, communities, and gaming.

## Features

### Social
- Posts with images and text
- Comments, replies, likes, shares
- Hashtags and mentions
- Friends and followers
- Direct messaging
- Search and explore

### Communities
- Public and private communities
- Role management
- Text channels
- Community settings

### Gaming
- 11 mini-games (Snake, 2048, Tic-Tac-Toe, etc.)
- XP and leveling system
- Achievements and badges
- Daily rewards
- Leaderboards
- Player statistics

### Shop
- Avatar customization
- Themes and backgrounds
- Badges and items

## Tech Stack
- HTML5
- CSS3 with Custom Properties
- Vanilla JavaScript (ES6+)
- IndexedDB for storage
- Service Worker for PWA

## Getting Started

1. Clone repository
2. Open index.html or use local server
3. Register account or use demo mode

## Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
